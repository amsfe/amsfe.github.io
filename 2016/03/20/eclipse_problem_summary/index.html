1. eclipse提示Note: This element neither has attached source nor attached Javadoc and hence no Javadoc could be found.解决方案：

(1).Android x.x.x（所用版本）documentation for Androidsdk（API开发文档）未安装：
adt-bundle-windows-x86_64-20140321\sdk\目录下缺少了个docs文档。于是每次鼠标停放在函数名上时，总提示：Note: This element neither has attached source nor attached Javadoc and hence no Javadoc could be found.可通过镜像网站下载安装包（关于如何通过镜像网站下载安装包，教程请参考↓↓↓）。

(2).API开发文档已安装：
解决方案请参考：http://blog.csdn.net/listening_music/article/details/6868076

2. Android SDK Manager 更新代理配置：

因为GFW的原因，国内使用SDK Manager下载会出现问题，可以使用镜像网站解决
镜像网站：http://mirrors.neusoft.edu.cn/
配置步骤：
(1).启动 Android SDK Manager ，打开主界面，依次选择「Tools」、「Options…」，弹出『Android SDK Manager - Settings』窗口；
(2).在『Android SDK Manager - Settings』窗口中，在「HTTP Proxy Server」和「HTTP Proxy Port」输入框内填入mirrors.neusoft.edu.cn和80，并且选中「Force https://… sources to be fetched using…」复选框。设置完成后单击「Close」按钮关闭『Android SDK Manager - Settings』窗口返回到主界面；

3.依次选择「Packages」、「Reload」。
