3.配置Git

为了把本地的仓库传到github，还需要配置ssh key

1.首先在本地创建ssh key

此处的your_email@youremail.com即为注册github时的邮箱。

2.添加ssh key到github

  • 打开本地C:\Documents and Settings\Administrator.ssh\id_rsa.pub文件

    此文件里的内容为刚才生成的密钥。默认为隐藏文件，需要手动显示。

  • 登录github系统。点击右上角的Account Settings -> SSH Public keys -> add another public keys

  • 把本地生成的密钥复制到里面（key文本框中），点击add key
